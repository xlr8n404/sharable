## Project Summary
Sharable (now Sharable) is a modern social media platform designed for seamless sharing. The platform features a minimalist, dark-themed aesthetic with a focus on speed and simplicity.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Fonts**: Syne (Google Fonts)
- **Deployment**: Turbopack (Next.js)

## Architecture
- `src/app/`: App router pages and layouts.
- `src/components/`: Reusable UI components.
- `src/lib/`: Utility functions and shared logic.

## User Preferences
- **Theme**: Pure black (dark) theme.
- **Brand Font**: Syne (always used for "Sharable" text).
- **Branding**: "Sharable" must always be written in this specific case format.

## Project Guidelines
- No comments unless requested.
- Use relative URLs for client-side API calls.
- Maintain a clean, minimalist design with distinctive typography and motion.

## Common Patterns
- Animations using `tailwindcss-animate` for smooth reveals.
- Centralized layout for landing pages.
